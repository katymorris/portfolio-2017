Stick To It

Visit Website

Stick To It is a gamification web app intended to encourage patients who are at high risk for HIV and other STD's to get tested. The user builds up points by taking Buzzfeed-style quizzes, setting doctor's appointments, and interacting with the interface, and are later able to redeem points for prizes when going to get tested at the clinic.

The app is built in Ruby on Rails using JavaScript and jQuery. We used Rails as an API and made various calls to JSON. I was directly and deeply involved throughout the development cycle by building the quiz page interface, building the splash page, assisting with the Dashboard interface and functionality, creating email functionality through Rails, and assisting with many other parts such as general debugging and adjusting the databases.

Healthcheckins

I'm currently working on Healthcheckins and expect to be finished by February 2017. Healthcheckins is a social media website created for UCLA as a research/test project. Users will be assigned to groups aligning with their health conditions such as diabetes or different types of cancer. Users can interact with mentors and other users to give and receive support for their conditions.

Healthcheckins is built in Ruby on Rails heavily relying on Rails as a frontend. All pages are hand coded and fully responsive. I built the Group page template with posts, comments, and likes and uses ajax calls to the controller for loading more posts (no reload), the Profile page template that features a listing of the user's interactions, Mentor dashboard that shows the mentor's patients and patient interactions, and all of the landing pages. I also set up all of the models, migrations, and relationships.
